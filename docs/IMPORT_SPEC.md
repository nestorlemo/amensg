# IMPORT_SPEC

- La importación será vía CSV (pendiente de implementación).
- El detalle del CSV debe almacenarse fila por fila.
- Las importaciones no deben eliminarse físicamente.
- MID no es globalmente único.
- Chip no es globalmente único.
- MID es único por `empresa_id + anio + mes`.
- Chip es único por `empresa_id + anio + mes`.
