# PRODUCT_SPEC

AMENSG is an activation billing system for monthly operation by company and partner.

This foundation phase creates the technical project base, documentation, local execution setup, Prisma model, seed data, Docker setup, and placeholder UI. It must not implement CSV import logic, full authentication, billing workflows, collection workflows, monthly close workflows, or partner settlement workflows yet.

## Core Modules

- Dashboard
- Importaciones
- Activaciones
- Facturacion
- Cobros
- Empresas
- Gastos
- Ingresos adicionales
- Liquidaciones
- Cierres
- Reportes
- Parametros
- Socios
- Usuarios
- Auditoria

## Stack

- Next.js
- TypeScript
- PostgreSQL
- Prisma
- Tailwind CSS
- shadcn/ui-ready configuration
- Docker
- Docker Compose
